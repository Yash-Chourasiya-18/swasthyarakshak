# SwasthyaRakshak+
> 🏥 Bharat ka Digital Health Companion — 57+ Features

A comprehensive healthcare web application prototype built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, works fully offline.

## 🌟 Features
- 🩺 **Health Trackers** — BP, SpO2, Sugar, Water, Sleep, Steps, BMI, Mood, Women Health, Vaccination
- 🚨 **Emergency & Safety** — Fall/Crash Detection, Voice SOS, Offline Emergency SMS, Medical ID, Emergency QR, Safe Zone, Ambulance Tracking
- 🤖 **AI & Smart** — Disease Risk, Diet Planner, Workout Plan, Voice Assistant, Medicine Interaction, Report Scanner, Symptom Checker, Fake Medicine Detector, Voice Notes
- 👨‍👩‍👧 **Family & Community** — Family Dashboard, Blood Donor Network, Health Challenges, Health Timeline
- 🏥 **Health Services** — Medicine Availability, Lab Booking, Home Nurse, Ambulance Live Tracking
- 🌾 **Rural India** — Low Internet Mode, 10 Regional Languages, Large Buttons, Offline Storage, Health Camps, ASHA Worker Mode

## 🚀 How to Run
Just open `index.html` in Chrome browser — no installation needed!

```bash
# Or run a local server:
npx serve . -l 3000
```

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Storage:** Browser localStorage (offline-first)
- **APIs:** Web Speech API, DeviceMotion API, Geolocation API, Camera API

## 📁 Structure
```
index.html          → Main dashboard
css/main.css        → Shared design system
js/main.js          → Shared utilities + sidebar
pages/              → 40+ individual feature pages
```

## 🇮🇳 Made for Bharat
Built with rural India, ASHA workers, and elderly users in mind.
